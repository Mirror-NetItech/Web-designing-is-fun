Let's break down the code step by step, explaining each section in detail:
  • *Style Section*: <style> .head::after{ content: ""; display: block; clear: both; } </style>
    This is a CSS stylesheet that targets an HTML element with the class head. The ::after pseudo-element is used to add content after the element.
    Note:
    • The content="" CSS property adds an empty content, which is necessary for the pseudo-element to work.
    • The display: block; CSS property with it's value makes the pseudo-element a block element, taking up the full width of its parent element.
    • The clear:both; CSS property with it's value clears both the left and right sides of the element, ensuring that no other elements are positioned next to it.
  • *Navigation Bar* This div element has the class nav-bar and several inline styles: Note:
    • The width: 95%; sets the width of the navigation bar to 95% of its parent element's width.
    • The float: left; makes the navigation bar float to the left, allowing other elements to position next to it.
    • The margin-left: 5px; adds a margin of 5 pixels to the left side.
    • The z-index: 999; sets the z-index (stacking order) to 999, ensuring that the navigation bar is on top of other elements.
    • The height: 10px; sets the height of the navigation bar to 10 pixels.
    • The padding: 0; removes padding from the navigation bar.
    • The margin-right: 0; removes margin from the right side.
    Hence div containing the class class="nav-bar" serves as a container for the link elements.
    Navigation Links: Each navigation link (a element) has several inline styles: - `text-decoration: none` removes underlines from the link text. - `color: white` sets the text color to white. - `list-style-type: none` removes bullet points from the link. - `font-weight: bolder` makes the font bold. - `border-radius: 20px 10px` adds a rounded corner effect to the link. - `background: blue` sets the background color to blue. - `onmouseover` changes the background color to light blue and text color to blue when hovered over.

    *Head Section*: This div element has the class "head" and several inline styles: - `background: greenyellow` sets the background color to green yellow. - `border-radius: 30px` adds a rounded corner effect. - `box-shadow: 2.5px 3.5px 5px` adds a box shadow effect.

    *Head Name and Logo* - `width: 50%` sets the width of the head name to 50% of its parent element's width. - `float: left` makes the head name float to the left. - `font-size: 105%` increases the font size of the head name to 105% of the default size.
    *Logo* - `width: 100%` sets the width of the head logo to 100% of its parent element's width.
    - `width: 50%` sets the width of the logo image to 50% of its parent element's width.
    - `margin-top: 10px` adds a margin of 10 pixels to the top of the logo image.

    *Main Section* This div element has the class "main" and several inline styles:
    - `background: lightseagreen` sets the background color to light sea green.
    - `border-radius: 25px` adds a rounded corner effect.
    - `position: center` centers the element (not supported in all browsers).
    `padding: 0` removes padding from the main section.
    `box-shadow: 2px 3px 4px` adds a box shadow effect.
    _Welcome Message and Skills Showcase_ - `font-size: 105%` increases the font size of the welcome message to 105% of the default size.
    - `margin-left: 15px` adds a margin of 15 pixels to the left of the welcome message.
